牌照架\License plate frame with a solgen
大吉大利今晚吃鸡 is “Winner winner, chicken dinner” translation in chinese 。（litterally：大吉大利=have a good luck, 今晚吃鸡=eat chicken tonight, 神器 = very useful object )
So，the meanning of solgen is “a very useful object which can help you got good luck to eat chicken tonight”。
